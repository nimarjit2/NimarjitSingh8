📊 Bibliometric Analysis using VOSviewer (Sample Dataset)

As part of my data analytics coursework, I performed a bibliometric analysis using VOSviewer to visualize relationships between research keywords and identify major research clusters. For this task, I used a sample bibliographic dataset provided for practice.

🔎 Steps involved in the analysis:

Importing the sample dataset into VOSviewer

Selecting co-occurrence analysis of keywords

Setting minimum occurrence thresholds

Generating a network visualization map

📈 Key insights:
The network visualization highlights different clusters of related keywords, showing how research topics are connected and grouped within the dataset.

💡 Key takeaway:
Bibliometric analysis helps researchers understand research trends, topic clusters, and relationships between academic concepts through visual network mapping.
